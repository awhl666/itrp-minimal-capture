# Downloads

Use this page to pick the correct jar for your Minecraft loader and version. The same files are available from the GitHub Release assets and from the repository `builds/` folder.

## Quick rules

- Fabric users should download a file with `fabric` in the name and the exact Minecraft version in the filename.
- NeoForge users should download a file with `neoforge` in the name and the matching `mc<version>` suffix.
- Do not install multiple loader variants at the same time.
- Put the selected jar in the target instance `mods` folder.
